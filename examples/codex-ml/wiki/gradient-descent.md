# 梯度下降 (Gradient Descent)

> Source(s): raw/simple-ml.md, raw/ch-01-Convolutional Neural Networks (CNNs), raw/ch-02-Recurrent Neural Networks (RNNs)

## 概述

梯度下降是一种迭代优化算法，通过向损失函数梯度的反方向更新模型参数来最小化损失函数。它是现代机器学习的基石，支撑从线性回归到深度神经网络的各种模型。

其核心思想是：给定一个函数 `J(θ)`，如何找到使 `J` 最小的参数 `θ`？算法通过重复执行与梯度负方向成比例的步骤来工作：

```
θ_{t+1} = θ_t - η ∇J(θ_t)
```

其中 `η`（eta）是学习率，一个控制步长的超参数。

## 变体

### 批量梯度下降（Batch Gradient Descent）

使用整个训练数据集计算梯度。这能提供真实梯度的准确估计，但对于大规模数据集而言计算成本高昂。每次参数更新需要处理所有 N 个训练样本，这使得它在数据集较大时变得不实用。

**优点**：收敛稳定，对于凸函数保证达到局部最小值。
**缺点**：大数据集不适用，每轮（epoch）进度慢。

### 随机梯度下降（Stochastic Gradient Descent，SGD）

每次迭代仅使用单个随机选择的训练样本来更新参数。这会在梯度估计中引入噪声，有助于跳出局部最小值，但会导致损失函数值波动。

```
θ_{t+1} = θ_t - η ∇J(θ_t; x_i, y_i)
```

**优点**：每次迭代速度快，适用于流式/在线数据，可跳出局部最小值。
**缺点**：更新方差大，可能需要学习率衰减（learning rate annealing）。

### 小批量梯度下降（Mini-batch Gradient Descent）

批量与随机之间的折衷：使用一小部分随机子集（mini-batch）的训练数据进行更新，通常包含 32-512 个样本。这是深度学习训练的主流方法和基石，在计算效率和梯度准确性之间取得平衡。由于能够利用 GPU 的向量化操作，它比纯 SGD 更稳定，同时比批量梯度下降显著更快。

**优点**：可利用 GPU 的向量化操作，比 SGD 更稳定，比批量梯度下降更快。
**缺点**：引入了一个新的超参数（batch size）。

## 高级优化器

### 动量（Momentum）

动量法通过累积历史梯度平滑更新方向，加速收敛并抑制振荡。其核心思想是：如果梯度方向在一段时间内保持一致，则应在该方向上积累动量，加快前进；如果梯度方向振荡（例如在峡谷两侧来回），动量则会抵消振荡，使更新路径更平滑。

动量法的更新规则如下：

```
v_t = γ v_{t-1} + η ∇J(θ_t)
θ_{t+1} = θ_t - v_t
```

其中，`v_t` 为速度向量，`γ`（通常为 0.9）控制动量衰减系数（又称阻尼系数）。`γ v_{t-1}` 项保留了历史梯度的方向信息，而 `η ∇J(θ_t)` 项引入了当前梯度的修正。相当于在每一步更新中，不仅考虑当前的梯度方向，还受到之前累积方向的影响。

动量有助于在损失表面的"峡谷"（即一处维度梯度方向急剧变化、另一处变化缓慢的狭窄区域）中导航，有效抑制振荡并加速收敛。可以直观理解为：类似一个球滚下山坡，它会累积惯性，从而平稳地穿过窄谷和小坑，不会因为局部的小凹凸而剧烈转向。

### RMSprop

RMSprop（Root Mean Square Propagation）为每个参数自适应调整学习率，方法是将梯度除以其近期幅度的运行平均值（即梯度平方的滑动平均）。这可以防止对于梯度持续较小的参数，学习率变得过小。其更新规则为：

```
E[g²]_t = β E[g²]_{t-1} + (1 - β) g_t²
θ_{t+1} = θ_t - (η / √(E[g²]_t + ε)) g_t
```

其中 `β` 是衰减率，`ε` 是一个小常数（如 1e-8）用于防止除零。由于每个参数都有自己独立的梯度平方累积值 `E[g²]`，那些频繁更新或梯度幅值较大的参数，其有效学习率会被自动降低；反之，那些更新稀疏或梯度较小的参数，其学习率则会保持或增加，从而确保所有参数都能以合适的步长更新。

### Adam（Adaptive Moment Estimation）

Adam 是一种结合了动量和 RMSprop 优点的自适应矩估计优化器，也是当前大多数深度学习框架的默认选择。它同时维护梯度的运行平均值（一阶矩）和梯度平方的运行平均值（二阶矩），并对早期步骤进行偏差校正：

```
m_t = β₁ m_{t-1} + (1 - β₁) g_t
v_t = β₂ v_{t-1} + (1 - β₂) g_t²
m̂_t = m_t / (1 - β₁ᵗ)
v̂_t = v_t / (1 - β₂ᵗ)
θ_{t+1} = θ_t - η m̂_t / (√v̂_t + ε)
```

默认超参数：`β₁ = 0.9`，`β₂ = 0.999`，`ε = 1e-8`。Adam 通过在每一步对一阶矩和二阶矩估计进行偏差校正，有效避免了早期训练阶段估计值偏零的问题。由于在广泛问题上的鲁棒性，Adam 是大多数深度学习框架中的默认优化器。

## 学习率调度

学习率 `η` 可以说是最重要的超参数。合适的调度策略能够显著影响训练效果。常见的调度策略包括：

- **步进衰减（Step decay）**：每 K 轮将 `η` 乘以一个衰减因子。
- **指数衰减（Exponential decay）**：`η_t = η_0 · exp(-kt)`。
- **余弦退火（Cosine annealing）**：沿余弦曲线平滑减小 `η`，通常带有热重启（warm restarts），使学习率周期性地从低到高再降低，有助于跳出局部极小点。
- **热身（Warmup）**：在训练的早期步骤中，将 `η` 从小值逐渐增加到目标速率。这对于 Transformer 等模型至关重要，可防止初始阶段因梯度过大而导致训练不稳定。

## 收敛性与挑战

梯度下降面临几个实际挑战：

1. **局部最小值和鞍点**：非凸损失表面（在深度学习中很常见）包含许多次优点。带有动量的现代优化器有助于跳出浅层局部最小值。
2. **梯度消失/爆炸**：在深层网络中，梯度可能会通过反向传播而指数级地缩小或增长。小心的权重初始化和归一化技术可以缓解这个问题。
3. **病态条件**：当 Hessian 矩阵的特征值范围很广时，损失表面会形成狭窄的"山谷"，从而减慢收敛速度。
4. **学习率的选择**：太大导致发散，太小导致收敛缓慢。Adam 等自适应方法降低了对这个选择的敏感性。

### 梯度消失与爆炸的深度分析

梯度消失与梯度爆炸是训练深层网络（特别是循环神经网络 RNN）时遇到的核心问题。当梯度通过时间反向传播（Backpropagation Through Time, BPTT）时，由于重复的矩阵乘法，梯度可能呈指数级缩小（消失）或增长（爆炸）。这使得网络难以学习长距离依赖关系——早期时间步的信息无法有效影响后续时间步的输出。

- **梯度消失**：梯度变得过小，导致网络权重几乎不更新，早期层无法学到有意义的特征。这在 RNN 训练长序列时尤为突出，因为信息在时间步之间传递时不断衰减。
- **梯度爆炸**：梯度变得过大，导致权重更新剧烈震荡甚至溢出（NaN）。梯度裁剪（Gradient Clipping）是应对梯度爆炸的简单有效方法：通过限制梯度向量的范数（如将梯度向量缩放到最大范数为 1.0 或 5.0），或将每个梯度分量限制在固定范围内，防止更新步长过大。
- **架构创新应对梯度消失**：梯度消失难以通过简单的超参数调整解决，通常需要架构层面的创新。例如，循环神经网络中的长短期记忆网络（LSTM）和门控循环单元（GRU）通过引入门控机制和独立的细胞状态路径，允许梯度以较小的衰减流经长时间步，从而有效缓解了梯度消失问题。

## 实践考虑

### 梯度裁剪（Gradient Clipping）

梯度裁剪是一种通过限制梯度范数来防止梯度爆炸的实用技术，特别适用于循环神经网络（RNN）等深度序列模型。当梯度向量的 L2 范数超过预设阈值时，将梯度按比例缩放至该阈值范围内，从而避免参数更新步长过大导致训练不稳定。

裁剪策略主要分为两类：
- **按范数裁剪（Norm clipping）**：计算整个梯度向量的范数，若超过阈值则按比例缩放。例如，设定最大范数为 1.0 或 5.0，将梯度向量 `g` 缩放为 `g · min(1, threshold / ||g||)`。
- **按值裁剪（Value clipping）**：将每个梯度分量独立限制在固定范围（如 `[-1, 1]`）内，直接截断超出范围的值。

按范数裁剪更为常用，因为它保持了梯度的方向信息，仅调整其幅度；而按值裁剪可能改变梯度的方向，但在某些场景下实现更简单。梯度裁剪是 RNN 训练的标准实践，常与截断反向传播（TBPTT）配合使用。

### 权重衰减（Weight Decay）

权重衰减（L2 正则化）是一种通过惩罚大权重来防止过拟合的正则化技术，与 Dropout 形成互补。它直接在参数更新步骤中对权重进行衰减，抑制权重增长以降低模型复杂度。

在标准 SGD 优化器中，权重衰减与向损失函数添加 L2 惩罚项在数学形式上完全等价；但在 Adam 等自适应优化器中，由于学习率会根据梯度历史进行自适应调整，直接在更新步骤应用权重衰减（即 AdamW 的做法）与将 L2 惩罚加入损失函数会产生细微但重要的差异。研究表明，解耦的权重衰减（decoupled weight decay）在自适应优化器中能带来更好的泛化性能。权重衰减通过限制参数范数的增长，有效降低了模型复杂度，是深度学习中广泛使用的正则化手段。

### Dropout

在训练过程中以概率 `p` 随机将神经元输出置为零（典型值为 0.5），迫使网络学习冗余表示，防止神经元之间的共适应。Dropout 常用于全连接层，是卷积神经网络的常用正则化技术，与权重衰减互补使用。对于 RNN，标准 dropout 会破坏循环动态，因此通常仅应用于非循环连接，或者使用在时间步上保持相同掩码的循环 dropout。

### 批归一化（Batch Normalization）

通过将激活值标准化为零均值和单位方差来稳定训练过程，减少内部协变量偏移（internal covariate shift）。批归一化通常在卷积层或全连接层之后应用，允许使用更高的学习率，并降低对权重初始化的敏感性。这一技术是 ResNet 等深层架构训练成功的关键，也是大多数计算机视觉任务的标准实践。

### 数据增强（Data Augmentation）

通过对训练数据进行随机变换（如随机裁剪、翻转、颜色抖动等）来人为扩大数据集规模，是一种强大的正则化手段，特别适用于卷积神经网络（CNN）等视觉模型。数据增强有助于提高泛化能力，减少过拟合。

### 早停法（Early Stopping）

早停法是一种简单有效的正则化策略，通过监控验证集上的损失（或评估指标），在模型性能停止改善时提前终止训练，从而防止过拟合。具体实现时，通常记录验证损失的最佳值，并设定一个容忍轮数（patience）：若验证损失在连续的 patience 轮内未再改善，则停止训练，并恢复验证损失最小时的模型参数。

早停法的核心直觉是：随着训练进行，训练损失通常会持续下降，但验证损失在达到某个最小值后可能开始上升，表明模型开始记忆训练数据中的噪声而非学习泛化规律。早停法通过截断训练过程，在模型泛化能力最佳时保留参数，相当于隐式地限制了模型的有效复杂度。该方法实现简单、计算开销低，且无需手动调整正则化强度，是深度学习训练中最常用的正则化技术之一，常与模型检查点（checkpointing）机制配合使用。

## 在卷积神经网络（CNN）中的应用

梯度下降是训练卷积神经网络的核心优化范式。CNN 通过权重共享和局部连接大幅减少了参数量，在保留空间结构信息的同时降低了优化难度，彻底改变了计算机视觉领域。

### CNN 架构核心组件

CNN 的核心构建模块包括：

- **卷积层（Convolutional Layers）**：应用可学习的滤波器（卷积核）在输入上滑动，通过计算点积生成特征图。一个大小为 `k × k × C_in` 的滤波器以步幅 `s` 在输入上滑动，在每个位置计算点积。同一层中的多个滤波器提取不同特征——从边缘、纹理到更高层次的抽象模式。
- **池化层（Pooling Layers）**：对特征图进行下采样以减少空间维度和计算成本。最大池化（取每个窗口中的最大值）是最常见的变体，提供一定程度的平移不变性；平均池化和全局平均池化等替代方案在特定架构中广泛应用。
- **激活函数**：ReLU（`f(x) = max(0, x)`）是隐藏层的标准选择，解决了困扰 sigmoid 和 tanh 的梯度消失问题。Leaky ReLU、PReLU 和 ELU 等变体为负输入提供小梯度，以解决"神经元死亡"问题。

### 训练实践

CNN 的训练通常采用基于动量的优化器（如 SGD with Momentum 或 Adam），并配合小批量梯度下降策略，以在计算效率和收敛稳定性之间取得平衡。具体而言：

- **权重初始化**：良好的初始化（如 He 初始化）对于深层 CNN 的训练至关重要，可避免初始阶段激活值饱和或梯度消失。
- **批归一化**：几乎已成为现代 CNN 架构的标准组件，通常应用于卷积层之后、激活函数之前。它通过标准化层间激活值的分布（零均值、单位方差），稳定了梯度流，显著放宽了对学习率和初始化策略的敏感性。
- **数据增强**：对于大部分视觉任务，数据增强（如随机裁剪、水平翻转、颜色抖动、旋转等）是等效于正则化的隐性约束，可帮助梯度下降找到泛化能力更强的解。从预训练模型进行迁移学习是大多数计算机视觉任务的标准做法。
- **权重衰减与 Dropout**：作为互补的正则化技术联合使用。权重衰减通过惩罚大权重抑制模型复杂度，Dropout 以概率 `p`（全连接层通常为 0.5）随机将神经元输出置零，迫使网络学习冗余表示。

### 对经典架构的影响

历史上的里程碑式卷积神经网络架构，都通过设计上的创新使梯度下降能够训练更深的网络：

- **LeNet-5（1998）**：最早的成功案例，为数字识别设计，建立了"卷积-池化-全连接"的标准流程，在小规模问题上验证了梯度下降在卷积结构中的有效性。
- **AlexNet（2012）**：将深度学习带入计算机视觉主流，以大幅优势赢得 ImageNet 竞赛。引入 ReLU 激活函数解决了梯度消失问题，并使用 Dropout 正则化，使梯度下降得以训练更大规模的网络。
- **VGGNet（2014）**：证明网络深度的重要性，采用小卷积核堆叠（3×3），梯度下降可稳定训练 16-19 层网络。其简洁性使其成为流行的基线和特征提取器。
- **ResNet（2015）**：引入跳跃连接（残差块），使梯度可以直接流向前层，有效缓解了深层网络中的梯度消失问题，从而可训练超过 100 层的网络。ResNet 的核心创新在于学习残差函数 `F(x) = H(x) - x` 而非直接拟合 `H(x)`，这种表征方式让梯度下降更容易找到合适的解。
- **EfficientNet（2019）**：利用神经架构搜索同时优化网络深度、宽度和分辨率，梯度下降在此基础上高效收敛。

## 在循环神经网络（RNN）中的应用

循环神经网络通过维护隐藏状态来处理序列数据，其基本递推关系为：

```
h_t = f(W_h h_{t-1} + W_x x_t + b)
```

其中 `h_t` 是时间步 `t` 的隐藏状态，`x_t` 是输入，`W_h`、`W_x` 是权重矩阵。

### 梯度消失与爆炸在RNN中的体现

训练 RNN 处理长序列时，梯度消失与爆炸问题尤为突出。由于梯度通过时间反向传播涉及重复的矩阵乘法（`W_h` 的幂次），梯度会迅速衰减或增长。这使得标准 RNN 难以学习序列中相距较远的依赖关系——例如，在语言模型中，一个句子开头的词难以影响句尾的预测结果。

### LSTM 与 GRU 的架构创新

为应对梯度消失，长短期记忆网络（LSTM）引入了门控机制来控制信息流，其中关键组件包括：
- **遗忘门（`f_t`）**：决定从细胞状态中丢弃哪些信息。
- **输入门（`i_t`）**：决定将哪些新信息存储到细胞状态。
- **输出门（`o_t`）**：决定基于细胞状态输出什么。
- **细胞状态（`c_t`）**：一条独立的路径，允许梯度以最小衰减流过长时间步。

门控循环单元（GRU）简化了 LSTM，将细胞状态与隐藏状态合并，仅使用重置门和更新门两个门控。GRU 通常能以更少的参数和更快的训练速度取得与 LSTM 相当的性能。

### 训练实践

RNN 的训练难度众所周知。截断反向传播（TBPTT）限制了梯度计算的时间步数，以计算可行性换取长程学习能力。教师强制（Teacher Forcing）在训练时用真实输出作为下一时间步的输入，加速收敛，但会导致训练与推理之间的差异，可通过计划采样（Scheduled Sampling）缓解。序列到序列模型（如编码器-解码器架构）曾主导机器翻译和摘要生成，但其将整个输入压缩为固定长度向量的瓶颈限制了长序列性能。

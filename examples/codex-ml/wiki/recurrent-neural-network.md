# 循环神经网络 (RNN)

> Source(s): raw/ch-02-Recurrent Neural Networks (RNNs)

循环神经网络（RNN）是一种处理序列数据的神经网络架构，其核心思想是通过维护一个**隐藏状态**来捕捉来自之前时间步的信息。基本递推关系为：

```
h_t = f(W_h h_{t-1} + W_x x_t + b)
```

其中 `h_t` 是时间 `t` 的隐藏状态，`x_t` 是输入，`W_h` 和 `W_x` 是权重矩阵，`b` 为偏置项，`f` 通常为非线性激活函数（如 tanh 或 ReLU）。

## 梯度消失与梯度爆炸问题

训练 RNN 处理长序列具有挑战性，因为通过时间反向传播（BPTT）的梯度倾向于**消失**（指数级变小）或**爆炸**（指数级变大）。这使得 RNN 难以学习长期依赖关系——早期时间步的信息无法有效影响后续时间步。

从数学上看，梯度随时间反向传播时会经历反复乘以权重矩阵 `W_h` 的过程。若 `W_h` 的特征值小于 1，梯度将随时间步呈**指数级衰减**；若特征值大于 1，则呈指数级增长。这种不稳定性是 RNN 处理长序列时的核心障碍。

- **梯度爆炸**：可以通过**梯度裁剪**（gradient clipping）来解决，即在反向传播时将梯度范数限制在一个阈值内，防止梯度值过大导致参数更新不稳定。梯度裁剪是 RNN 训练中的标准实践，通过限制梯度范数确保优化过程的数值稳定性。
- **梯度消失**：需要架构层面的创新来克服，主要依靠引入门控机制。

## LSTM 与 GRU

为了解决梯度消失问题，研究者提出了门控机制。

### 长短期记忆网络 (LSTM)

LSTM 引入了一个**细胞状态** (`c_t`)，作为一条独立的路径，允许梯度以最小衰减流动。它通过三个门控制信息流，使网络能够灵活地决定何时丢弃旧信息、何时存储新信息以及何时输出当前状态：

- **遗忘门** (`f_t`)：决定从细胞状态中丢弃什么信息。
- **输入门** (`i_t`)：决定存储什么新信息。
- **输出门** (`o_t`)：决定基于当前细胞状态输出什么。
- **细胞状态** (`c_t`)：作为信息流的独立通道，梯度可以在此路径上几乎无衰减地传播，从而缓解长期依赖问题。

### 门控循环单元 (GRU)

GRU 是 LSTM 的简化版本，它将细胞状态和隐藏状态合并为一个单一的隐藏状态，仅使用两个门控机制——**重置门**（reset gate）和**更新门**（update gate）——来控制信息流：

- **重置门**：决定如何将新的输入与之前的记忆相结合，控制忽略过去隐藏状态的程度。
- **更新门**：决定前一时刻的隐藏状态有多少信息被保留到当前时刻，同时控制新候选隐藏状态的融入比例。

通过这种合并设计，GRU 在保留 LSTM 处理长期依赖能力的同时，显著减少了参数量。GRU 通常以更少的参数和更快的训练速度达到与 LSTM 相当的性能，因此在计算资源受限或需要更快训练的场景中更具优势。实验研究表明，GRU 与 LSTM 在不同任务上各有优劣，选择哪种架构往往取决于具体应用场景和计算约束。

## 序列到序列模型

编码器-解码器架构是 RNN 时代的核心范式。**编码器**将输入序列处理成一个固定长度的上下文向量，然后**解码器**基于该向量生成输出序列。该架构曾主导机器翻译、摘要生成和对话系统，是 Transformer 模型兴起前的主要序列建模方案。

**关键局限**：将整个输入序列压缩成一个固定长度的向量会形成**信息瓶颈**（information bottleneck），特别是在处理长序列时，压缩过程会丢失大量细节信息，导致性能下降明显。这一局限性直接催生了注意力机制的发展。

## 训练 RNN

训练 RNN 以困难著称。常用技术包括：

- **截断的时间反向传播 (TBPTT)**：限制反向传播的时间步数，在计算可行性与长期学习能力之间取得平衡。通过只对最近若干时间步进行反向传播，控制了计算复杂度和内存开销，是训练 RNN 的标准实践。

- **教师强制 (Teacher Forcing)**：在训练时将真实输出（ground-truth）作为下一时间步的输入喂给解码器，而非使用模型自身的预测，从而加速收敛。然而，这一策略会导致**训练-推理不匹配**（train-inference discrepancy）问题：训练时模型始终接触真实输出，而推理时则必须依赖自身生成的预测，一旦产生错误便会逐步累积放大。

- **计划采样 (Scheduled Sampling)**：为缓解教师强制导致的训练-推理不匹配问题，计划采样在训练过程中逐步使用模型自身的预测替代真实输出作为下一时间步的输入。通过设定一个衰减策略（如线性衰减或指数衰减），训练初期主要依赖真实输出以保证稳定学习，后期逐渐增加模型预测的比例，使模型适应推理时的分布，从而缩小训练与推理之间的差异。

- **正则化**：由于标准 Dropout 会破坏循环动力学（在不同时间步使用不同的 Dropout 掩码会干扰隐藏状态的传递），通常将 Dropout 应用于**非循环连接**。**循环 Dropout**（recurrent dropout）是 RNN 特有的正则化技术，它在所有时间步使用**相同的 Dropout 掩码**，从而保持循环结构的一致性，避免破坏跨时间步的隐藏状态传递。此外，**权重绑定**（weight tying）通过在嵌入层和输出层之间共享权重矩阵，有效减少模型参数量并提升泛化能力，是 RNN 训练中另一项重要的正则化手段。
